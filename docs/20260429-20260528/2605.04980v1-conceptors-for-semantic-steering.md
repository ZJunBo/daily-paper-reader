## 摘要
基于激活的引导可以在推理时控制大语言模型的行为，但主流范式将每个概念简化为单一方向，其几何结构基本未被探究。我们并非选择单一引导方向，而是使用概念器：从双极概念的两极激活池中估计的软投影矩阵，保留了概念的多维子空间。几何分析表明，双极子空间严格包含了单向量基线。我们进一步证明，概念器配额提供了无参数的层选择诊断，在三个指令微调模型和三个语义维度上，预测概念可分离性的皮尔逊相关系数高达r=0.96。除选择外，概念器还支持闭合形式的布尔代数（与、或、非）：我们在主题相关的子概念上评估了概念器的组合性。通过系统的五轴设计空间评估，概念器在概念子空间是多维的层上匹配或优于加性基线，同时产生显著更少的退化输出。概念器引导是一种几何上有原则、可组合且实际更安全的替代方案，替代从有限对比对中获得的单方向引导。

## Abstract
Activation-based steering provides control of LLM behavior at inference time, but the dominant paradigm reduces each concept to a single direction whose geometry is left largely unexamined. Rather than selecting a single steering direction, we use conceptors: soft projection matrices estimated from activations pooled across both poles of a bipolar concept, which preserve the concept's full multidimensional subspace. A geometric analysis shows the bipolar subspace strictly subsumes the single-vector baseline. We further show that the conceptor quota provides a parameter-free layer-selection diagnostic, predicting concept separability with Pearson correlations up to r=0.96 across three instruction-tuned models and three semantic dimensions. Beyond selection, conceptors admit a closed-form Boolean algebra (AND, OR, NOT): we evaluate conceptor compositionality on thematically related sub-concepts. Across a systematic five-axis design-space evaluation, conceptors match or outperform additive baselines at layers where concept subspaces are multi-dimensional while producing substantially fewer degenerate outputs. Conceptor steering is a geometrically principled, compositional, and practically safer alternative to single-direction steering from a limited number of contrastive pairs.

## 1. 论文的核心问题与整体含义（研究动机和背景）

现有基于激活的转向（activation-based steering）方法（如激活添加、对比激活添加等）将每个语义概念简化为**单一方向向量**，忽略了概念在隐状态中实际占据的**多维子空间结构**。这种简化导致两个问题：
- **表达能力受限**：单向量无法捕捉概念的完整几何结构，丢失了残差结构中的有用信息。
- **层选择成本高**：通常需要逐层训练探测分类器（probe）来选择干预层，代价随模型深度和概念数量线性增长。

论文提出使用**Conceptor（概念器）**——一种软投影矩阵，从双极概念（如正/负情感）的激活中学习完整的子空间，并通过其内置的布尔代数（AND/OR/NOT）实现可组合的、几何上有原则的推理时控制。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 2.1 核心思想
- **Conceptor 矩阵 C**：定义为最小化重构误差的优化问题的闭式解：
  \[
  C = \arg\min_C \frac{1}{N}\sum_{i} \|x_i - C x_i\|^2 + \alpha^{-2}\|C\|_F^2
  \]
  其中 \( R = \frac{1}{N}X^\top X \) 为样本相关矩阵，解为 \( C = R(R + \alpha^{-2} I)^{-1} \)。

- **软投影解释**：在 R 的特征基下，C 的对角元素为 \( \gamma_j = \sigma_j / (\sigma_j + \alpha^{-2}) \)，即信号能量高的方向被保留（\(\gamma_j \to 1\)），低能量的方向被抑制（\(\gamma_j \to 0\)），\(\alpha\) 控制孔径（分辨率）。

- **双极训练**：将正负两极的激活合并到一个相关矩阵中训练 Conceptor，而非只使用单极。几何分析表明，双极子空间捕获了单极均差向量 95% 以上的能量。

### 2.2 关键技术细节
- **Conceptor 配额 q(C) = tr(C)/d**：软有效维度，整合了整个谱的信息，可无监督预测层间概念可分离性（Pearson r 可达 0.96），替代需标签的探测分类器。
- **布尔代数**：支持闭式逻辑运算——NOT(¬C = I - C), AND(C_A ∧ C_B = (C_A^{-1} + C_B^{-1} - I)^{-1}), OR（通过德摩根律）。无需原始激活数据即可组合多个概念。
- **干预方式**：
  - 替换（Replace）：\( z' = \beta C z \)
  - 插值（Interpolation）：\( z' = (1-\beta)z + \beta C z \)

### 2.3 算法流程（文字说明）
1. 收集 100 个对比句对，提取每一层的隐状态（残差流或注意力输出）。
2. 计算相关矩阵 R，解出 Conceptor C（需选 α）。
3. 在生成时作为前向钩子，在选定层将隐状态按替换或插值方式修改，β 控制强度。
4. 支持多概念组合：先计算布尔运算后的新 Conceptor，再应用。

## 3. 实验设计：数据集、基准与对比方法

### 3.1 使用数据集
- **概念**：情感（sentiment）、政治倾向（political leaning）、抑郁（depression）。
- **外部探针数据集**（用于层选择诊断）：SST-2、Rotten Tomatoes、TweetEval（情感）；Hyperpartisan News（政治）；Reddit Depression、Clinical Depression（抑郁）。

### 3.2 基准与对比方法
- **Additive baselines**：
  - Addition：\( z' = z + \beta \bar{v} \)，\(\bar{v}\) 为所有训练句子的平均激活。

- **模型**：Gemma-2-2B-IT、Gemma-2-9B-IT、Qwen-2.5-3B-IT（均为指令调优模型）。

### 3.3 评估指标
- **胜率（Win Ratio）**：引导输出在目标维度上得分高于基线的比例（使用外部分类器打分）。
- **退化输出检测**：引导输出的平均长度 > 2 倍基线长度时视为退化（主要失败模式）。
- **MCQ 概率与选择率**：通过 A/B/C/D logits 读取模型对各类别的偏好。

## 6. 论文的主要结论与发现

1. **双极子空间优越性**：双极训练的子空间捕获了均差方向 ≥95% 的能量，而单极仅 ~10%。Conceptor 可复现加性转向的效果，同时保留更多结构信息。
2. **Conceptor 配额可无监督选择层**：r(Quota, AUC) 可达 0.96，优于或等于 EVR，且无需标签。
3. **跨概念重叠低**（0.18-0.38），遵循语言上合理的层级（情感-政治 < 情感-抑郁 < 政治-抑郁），解释了布尔运算的行为：AND 需要主题相关性，NOT/OR 在低重叠时仍有效。
4. **Conceptor 匹配或超越加性基线**：在多维子空间的层上胜率更高（0.70-0.82 vs 0.50-0.62），且退化输出率显著更低（13% vs 58%）。
5. **布尔组合性有效**：AND-NOT 能选择性增强/抑制特定概念，尤其在重叠较高的概念对（堕胎×LGBTQ）上效果明显。
6. **设计空间建议**：推荐残差流放置、全 token 干预、替换操作（但注意安全性）或插值（安全优先）、任何训练源、持续干预。

## 7. 优点：方法或实验设计上的亮点

- **几何上有原则**：基于二阶统计量（相关矩阵）的软投影，而非简单的向量加减。
- **无参数层选择工具**：Conceptor 配额可在无标签情况下高效诊断概念可分离性，省去训练探测分类器的成本。
- **闭式布尔代数**：无需原始数据即可组合概念，这是单方向方法无法直接做到的。
- **系统性设计空间分析**：首次系统评估 5 个关键设计因素，提供实用的工程指南。
- **多维度评估**：既用开放生成（胜率），又用封闭 MCQ（logit 概率），避免单一评估范式偏差。
- **退化输出检测机制**：基于长度比检测模型崩溃，方法简单有效。
- **实验覆盖三个不同模型家族**（Gemma-2B/9B，Qwen-3B），增强结论泛化性。
